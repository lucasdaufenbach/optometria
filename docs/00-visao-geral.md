# 00 — Visão Geral

## O que é

Sistema web para gestão completa de uma clínica de optometria.

Duas aplicações em um só projeto:

- **Área do Paciente** — autoagendamento de consultas. Mobile first, simples, poucos passos. Público idoso, pouca familiaridade com tecnologia.
- **Área Administrativa** — gestão da clínica pelo optometrista. Estilo ERP: grids, filtros, dashboards, atalhos. Uso em computador.

## Objetivo

Substituir planilhas e controles manuais por um software profissional, estável e fácil de manter por muitos anos.

Não é um MVP descartável. É um produto construído de forma incremental, em pequenas etapas, mantendo o projeto sempre funcionando.

## Funcionalidades-alvo (visão completa)

- Gestão completa de pacientes
- Agenda inteligente (visão semanal e mensal)
- Autoagendamento pelo paciente
- Histórico clínico
- Cadastro de consultas e tipos de consulta
- Upload de imagens/arquivos
- Dashboard gerencial
- Integração com WhatsApp e mensagens automáticas
- Relatórios
- Configurações
- Controle administrativo

## Princípios resumidos

Clareza > esperteza. Poucas dependências. Organização por domínio. Crescimento gradual. Cada sessão entrega algo utilizável e deixa o projeto compilando.

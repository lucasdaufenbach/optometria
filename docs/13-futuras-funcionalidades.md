# 13 — Futuras Funcionalidades

Ideias além do roadmap atual. Aqui ficam guardadas sem comprometer o foco. Promover para [10-backlog.md](10-backlog.md) / [09-roadmap.md](09-roadmap.md) quando amadurecerem.

## Escala

- Múltiplos profissionais (agendas separadas).
- Múltiplas clínicas (multi-tenant leve via `clinica_id`).
- Múltiplas agendas por profissional.

## Comunicação

- Lembretes automáticos de consulta (WhatsApp).
- Confirmação/cancelamento pelo paciente via mensagem.
- Pesquisa de satisfação pós-consulta.

## Clínico

- Modelos de receita/laudo.
- Comparativo de exames ao longo do tempo.
- Galeria de imagens por paciente.

## Gestão

- Relatórios financeiros e de produtividade.
- Indicadores de faltas/no-show.
- Exportação de dados (CSV/Excel).

## Plataforma

- Migração de armazenamento local → S3 em produção.
- Notificações por e-mail.
- Logs/auditoria de ações administrativas.
- Backup automatizado.

> Nada aqui é compromisso. É memória de ideias para não se perderem.
